## FNF Play Engine
![Play Engine logo](logo.png)
- Play Engine is FNF Engine what adds new features to original Friday Night Funkin' game.
### Features
- Updated HUD
- New FPS Counter
- hxCodec Videos Support
- Android Support
- And Many Different Cool Things
